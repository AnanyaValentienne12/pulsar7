pulsar — market anomaly detector
A real-time crypto signal scanner that watches market data across venues, flags anomalies, and shows the status of a multi-agent intelligence layer.
Built to understand how signal ingestion and anomaly detection work in autonomous trading systems.
What it does

Scans simulated live market data across 7 crypto assets
Flags anomalies (funding rate spikes, liquidation cascades, correlation breaks)
Shows the status of 7 specialized agents (Signal → Research → Strategy → Risk → Execution → Learn → Synthesis)
Updates in real time

Why I built this
I wanted to understand how the first layer of an autonomous market system actually works — what gets flagged, why, and how it hands off to the next agent in the chain.
This is a prototype/learning project. Not financial advice.
